# Food Order App

## 📌 概要
Mock API からデータを取得し、カート管理・注文フォーム入力まで実装したフード注文アプリです。React の基礎〜応用を統合した集大成的プロジェクトです。

## 🛠️ 技術スタック
- React
- Context API
- Fetch API
- Form handling

## 🎯 主な機能
- 食品一覧の取得・表示
- カートへの追加、数量調整、価格計算
- 注文フォーム入力 → 注文完了フロー
- モーダル内での画面切替

## 💡 学習ポイント
- Mock API・fetch によるデータ取得
- Context によるアプリ全体状態管理
- フォームデータの取り扱い
- 多段階 UI フローの設計
